# Changelog

## 0.1.0

* Add image package. Supports limited set of options:
	* `crop-h`
	* `crop-w`
	* `crop-x`
	* `crop-y`
	* `format`
	* `height`
	* `width`
* Add pdf package. Supports limited set of options:
	* `margin-top`
	* `margin-bottom`
	* `margin-left`
	* `margin-right`
	* `page-height`
	* `page-width`
	* `page-size`
